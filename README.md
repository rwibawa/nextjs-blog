# nextjs-blog
* [Tutorial](https://nextjs.org/learn/basics/create-nextjs-app)

## 1. Setup
```sh
$ npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn-starter/tree/master/learn-starter"

$ cd nextjs-blog
$ npm run dev
```
